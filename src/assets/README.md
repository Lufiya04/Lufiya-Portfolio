# Assets

Place the following files in this directory:

- `profile.jpg` — Your profile photo (used in the Hero section)
- `resume.pdf` — Your CV/Resume (place in the `public/` folder at the project root, not here)

## Profile Image
The Hero component imports: `import profileImg from "@/assets/profile.jpg"`
Add your photo here as `profile.jpg`.

## Resume
The Navbar and Hero both link to `/resume.pdf`.
Place your resume PDF at: `public/resume.pdf`
